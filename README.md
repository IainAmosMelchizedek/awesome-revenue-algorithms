# Awesome Revenue Algorithms

> Battle-tested algorithms, libraries, and techniques for **monetization optimization**, dynamic pricing, revenue management, churn reduction, and systems that actually move the needle.

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

## Dynamic / Algorithmic Pricing

- **[PuLP](https://coin-or.github.io/pulp/)** — Linear programming solver for optimal pricing and resource allocation. Widely used in production revenue management. (Python)
- **[OR-Tools](https://developers.google.com/optimization)** — Google's powerful constraint programming suite for complex pricing and yield optimization. (Python, C++, Java)
- **[Nevergrad](https://facebookresearch.github.io/nevergrad/)** — Derivative-free optimization library from Facebook Research. Excellent for black-box pricing experiments. (Python)
- **[contextualbandits](https://github.com/david-cortes/contextualbandits)** — Efficient contextual bandit algorithms (LinUCB, etc.) for personalized real-time pricing. (Python)
- **Greedy Pricing Heuristics** — Simple, fast, interpretable rules with proven revenue lift. *Murphy Street production example*: Optimized pricing decisions leading to increased monetization.
- **[Dynamic Pricing with DQN](https://github.com/Hossein-Alibandehloo/Dynamic-Pricing-with-Reinforcement-Learning)** — Deep Q-Network reinforcement learning approach for dynamic pricing. (Python)
